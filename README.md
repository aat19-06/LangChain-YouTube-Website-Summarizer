# 🌐 AI YouTube & Website Summarizer

An AI-powered web application that generates concise summaries from **YouTube videos** and **web pages** using **LangChain**, **Groq LLMs**, and **Streamlit**.

Simply paste a YouTube or website URL, and the application extracts the content before producing an accurate summary in seconds.

---

## ✨ Features

- 🎥 Summarize YouTube videos
- 🌍 Summarize website articles
- 🤖 Powered by Groq LLMs
- 🔗 Automatic URL validation
- ⚡ Fast AI-generated summaries
- 💻 Interactive Streamlit interface

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/yt-website-summarizer-langchain.git
```

Move into the project directory

```bash
cd yt-website-summarizer-langchain
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```bash
streamlit run app.py
```

---

## 🔑 Configuration

Enter your **Groq API Key** in the sidebar after launching the application.

Then paste either:

- A YouTube video URL
- A Website URL

Click **Summarize** to generate the summary.

---

### Output

- AI-generated summary in approximately 300 words
